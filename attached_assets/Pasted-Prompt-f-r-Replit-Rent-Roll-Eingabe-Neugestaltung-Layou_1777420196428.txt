Was passiert nach Ablauf der Laufzeit
Optionen: "Neuvermietung" | "Leerstand"
Default: "Neuvermietung"
Bei Neuvermietung: Nach Ablauf der Laufzeit wird 1 Monat Leerstand angesetzt, dann läuft die Miete weiter zur letzten indexierten Miete (selbe Konditionen, unbefristet)
Bei Leerstand: Nach Ablauf der Laufzeit fällt die Miete komplett weg, kein Mieteinfluss mehr im Cashflow

Indexierung % (ersetzt Indexierungsintervall + Indexierungssatz):

Ein einziges Number-Input-Feld
Prozentsatz der in Market Assumptions angegebenen Opex Inflation (NICHT Capex Inflation — Korrektur: es ist die allgemeine Inflationsrate/VPI, die als Referenz dient)
Beispiel: Wenn Opex Inflation in Market = 2,5% und Indexierung = 80%, dann steigt die Miete jährlich um 2,5% × 80% = 2,0%
Default: 100% (volle Indexierung)
Die Miete erhöht sich jährlich um diesen berechneten Satz
Tooltip oder Hilfetext: "Anteil der Inflation (aus Market) der auf die Miete angewendet wird"

Entfernte Spalten:

Mietende → ersetzt durch Laufzeit (Monate)
Indexierungsintervall → entfällt, Indexierung ist immer jährlich
Indexierungssatz  → ersetzt durch Indexierung % (Anteil der Inflation)

5. Auswirkung auf Cashflow-Berechnung
Die monatliche Cashflow-Berechnung in propertyCashFlowModel.ts muss diese Logik abbilden:
typescript// Für jede Einheit in jedem Monat:

function computeUnitRentForMonth(unit: RentRollUnit, monthIndex: number, acqDate: Date, opexInflation: number): number {
  const currentDate = new Date(acqDate);
  currentDate.setMonth(currentDate.getMonth() + monthIndex);
  
  const leaseStart = new Date(unit.leaseStart);
  const leaseEndDate = new Date(leaseStart);
  leaseEndDate.setMonth(leaseEndDate.getMonth() + unit.leaseDurationMonths);
  
  // VOR Mietstart: keine Miete
  if (currentDate < leaseStart) return 0;
  
  // WÄHREND Laufzeit: Miete mit jährlicher Indexierung
  if (currentDate <= leaseEndDate) {
    const yearsSinceStart = (currentDate.getTime() - leaseStart.getTime()) / (365.25 * 86400 * 1000);
    const annualIncrease = opexInflation * unit.indexationPercent / 100 / 100;
    // Indexierung wird nur auf abgeschlossene volle Jahre angewendet
    const completedYears = Math.floor(yearsSinceStart);
    return unit.monthlyRent * Math.pow(1 + annualIncrease, completedYears);
  }
  
  // NACH Laufzeit:
  if (unit.leaseEndAction === 'Leerstand') {
    // Leerstand: keine Miete mehr
    return 0;
  }
  
  // Neuvermietung: 1 Monat Leerstand, dann weiter zur letzten indexierten Miete
  const monthsAfterEnd = Math.round((currentDate.getTime() - leaseEndDate.getTime()) / (30.44 * 86400 * 1000));
  
  if (monthsAfterEnd <= 1) {
    // Leerstandsmonat
    return 0;
  }
  
  // Ab Monat 2 nach Ende: Weiterführung mit letzter indexierter Miete
  // Letzte indexierte Miete = Miete am Ende der Laufzeit
  const yearsOfOriginalLease = unit.leaseDurationMonths / 12;
  const completedYearsAtEnd = Math.floor(yearsOfOriginalLease);
  const annualIncrease = opexInflation * unit.indexationPercent / 100 / 100;
  const rentAtLeaseEnd = unit.monthlyRent * Math.pow(1 + annualIncrease, completedYearsAtEnd);
  
  // Weiter indexieren ab Neuvermietung
  const yearsAfterRenewal = (monthsAfterEnd - 1) / 12;
  const completedYearsAfterRenewal = Math.floor(yearsAfterRenewal);
  return rentAtLeaseEnd * Math.pow(1 + annualIncrease, completedYearsAfterRenewal);
}

6. WALT-Berechnung anpassen
WALT berechnet sich jetzt aus Mietstart + Laufzeit statt aus Mietende:
typescriptfunction computeWALT(units: RentRollUnit[], referenceDate: string): number {
  const ref = new Date(referenceDate);
  let weightedSum = 0;
  let totalArea = 0;
  
  units.forEach(u => {
    if (u.tenant && u.leaseStart && u.leaseDurationMonths > 0) {
      const leaseEnd = new Date(u.leaseStart);
      leaseEnd.setMonth(leaseEnd.getMonth() + u.leaseDurationMonths);
      const remainingYears = Math.max(0, (leaseEnd.getTime() - ref.getTime()) / (365.25 * 86400 * 1000));
      weightedSum += u.area * remainingYears;
      totalArea += u.area;
    }
  });
  
  return totalArea > 0 ? weightedSum / totalArea : 0;
}
